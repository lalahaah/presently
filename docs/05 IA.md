## **2\. IA Hierarchy Chart**

### **Level 1: Root / Main Navigation**

1. **Home (Dashboard)**  
2. **Recipients (Gift CRM)**  
3. **Settings (Profile & Pro)**

### **Level 2: Detailed Screen & Content Mapping**

#### **1\. Home Module (The Hub)**

* **1.1. Header**: User Greeting & Pro Status Badge.  
* **1.2. Primary CTA**: "Find a Gift" (Starts Step-based Profiler).  
* **1.3. Upcoming Events Widget**:  
  * Upcoming birthdays/anniversaries from saved recipients.  
  * "Start Recommendation" quick link per event.  
* **1.4. Recent Recommendations**:  
  * Horizontal scroll of the last 3 curated results.  
* **1.5. Ad Placement**: Bottom Static Banner.

#### **2\. Gift Profiler Module (Linear Flow)**

* **2.1. Step 1: Relationship**:  
  * Grid of relationship icons (Spouse, Friend, etc.).  
* **2.2. Step 2: Personality & Interests**:  
  * Selectable Chip tags (Searchable/Expandable).  
  * Custom interest input field.  
* **2.3. Step 3: Context & Occasion**:  
  * Occasion dropdown (Birthday, Just because, etc.).  
  * Story-telling input (Free-form text).  
* **2.4. Step 4: Budget & Fine-tuning**:  
  * Price Range Slider ($).  
  * "Include/Exclude" toggle (e.g., No alcohol, Eco-friendly).  
* **2.5. Processing**:  
  * AI Analysis Animation.  
  * **Full-screen Interstitial Ad (Trigger point).**

#### **3\. Recommendation Results (The Outcome)**

* **3.1. Recommendation Gallery**:  
  * Swipeable Result Cards (Top 3-5 picks).  
* **3.2. Card Content**:  
  * Item Photo, Title, Price indicator ($$$).  
  * **AI Reasoning**: "Why it's perfect".  
  * **AI Greeting**: "What to say".  
* **3.3. Action Bar**:  
  * "View on Amazon" (Affiliate Link).  
  * "Save Recipient & Event" (Entry to CRM).  
  * "Share/Copy" result.

#### **4\. Recipients Module (Gift CRM)**

* **4.1. Recipient List**:  
  * Searchable list of saved people.  
  * Display: Name, Relationship, Days until next event.  
* **4.2. Recipient Detail**:  
  * Stored traits/interests.  
  * Past gift history (Logs).  
  * Edit/Delete options.  
* **4.3. Add New Recipient**: Manual entry form.

#### **5\. Settings Module**

* **5.1. Account**: Email, Social Link, Sign Out/Delete Account.  
* **5.2. Subscription (Go Pro)**:  
  * Tier comparison (Free vs. Pro).  
  * Restore Purchase.  
* **5.3. Preferences**:  
  * Push Notification settings (Reminder days).  
  * Theme (Light/Dark).  
* **5.4. Legal & Support**:  
  * Privacy Policy, Terms of Service, Contact Support.

## **3\. Data & Interaction Logic**

| Node | Input Data | Output / Outcome | Tech Stack Note |  
| Profiler | Traits, Context, Budget | JSON for AI | Supabase Edge Function |  
| Results | AI Generated JSON | UI Rendered Cards | GPT-4o-mini |  
| CRM | User Input / AI History | PostgreSQL Rows | Supabase DB |  
| Monetization | User Tier Status | Ads or Buy Link | AdMob / Amazon |
